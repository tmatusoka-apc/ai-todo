# 利用手順書（AI-Sync Health / AI Todo）

エンドユーザー向けの操作ガイドです。社内の健康習慣化・減量サポートのために、本アプリの各機能をどう使うかをまとめています。開発・運用に関する技術情報は [README](../README.md) を参照してください。

---

## 1. ログイン

1. アプリのURL（CloudFront配信URL）にアクセスします。
https://d33o0pb58hdrlj.cloudfront.net/index.html
2. ログイン画面が表示されたら、Amazon Cognito Hosted UI で社内アカウントにログインします。
3. 初回ログイン時はプロフィール（氏名・目標など）の設定を行ってください（サイドバー「⚙️ プロフィール」）。

---

## 2. Health Todo（メイン機能）

ログイン後、サイドバーの各メニューから以下の機能を利用できます。

### 📊 ダッシュボード
- 今週のスコア（ポイント・レベル）、カテゴリ別（運動／食事／睡眠）ポイントを確認できます。
- AIが分析した「本日のAI推奨タスク」が表示されます。達成率に応じて毎日内容が更新されます。

### ✓ タスク
- 当日のタスク一覧の確認・完了チェックができます。
- 「💡 AIからの提案タスク」をそのまま自分のタスクとして追加できます。
- 新規タスクは手動でも追加可能です（カテゴリ・ポイントを指定）。

### 🔥 習慣
- 運動・食事・睡眠などの習慣をトラッキングし、達成日数（ストリーク）を管理します。
- 「習慣を完了」を押すとストリークが更新され、ポイントが加算されます。

### 👥 グループ
- 「グループを作成」で新しいグループを作成できます。
- 既存グループに参加する場合は、グループ作成者から **参加ID** を共有してもらい、「グループに参加」画面でIDを入力してください。
  - 参加IDは、自分が作成・参加しているグループカードの「参加ID」欄からいつでもコピーできます（「コピー」ボタン）。
- グループ内では週間ランキング（ポイント順）と活動フィード（メンバーの達成をリアクションで称賛できる）が確認できます。

### 📈 分析
- 習慣のパフォーマンス分析、AIによる達成傾向のレポート、タスク・グループ単位の詳細分析を確認できます。

### ⚙️ プロフィール
- 氏名・基本設定・目標（減量目標、トレーニング目標など）を管理します。
- 「🎯 目標管理（逆算プラン）」で目標を登録すると、AIが目標達成までの**週ごとの行動計画（バックキャスト）**を自動生成します。
- 獲得済みバッジの確認、ログアウトはこのページから行います（サイドバーにログアウトボタンはありません）。

### 🔐 管理ダッシュボード（admin / analyst ロールのみ）
- 該当ロールを持つユーザーのみサイドバーに表示されます。組織全体の健康状態・達成率をAIが分析して可視化します。
- 直接アクセスする場合は `/admin-analysis.html` です。

## 3. 減量サポートサイト（weight-loss.html）

サイドバーの「⚖️ 減量サポート」リンクから移動できます。Health Todo に戻る場合は、減量サポートサイトのサイドバーにある「🏠 Health Todoへ戻る」リンクを使用してください。

### 📊 ダッシュボード
- 減量の進捗（ストリーク、本日のToDo進捗、過去30日間の体重推移グラフ）を確認できます。

### ⚖️ 体重ログ
- 日々の体重を記録すると、BMIが自動計算されグラフに反映されます。
- 記録の削除も可能です。

### ✅ 減量ToDo
- 難易度別のスターターキットからタスクを選んで追加できます。
- タスク作成時に **「🔄 定期タスク」** にチェックを入れると、完了した翌日に自動で未完了状態にリセットされます（毎日続ける習慣タスク向け）。
- チェックを入れない場合は **不定期タスク** として扱われ、完了後もそのまま完了状態を維持します（一度きりのタスク向け）。

### 📚 ナレッジベース
- ダイエット・トレーニングに関する個人のメモ（ナレッジ）を登録できます。
- 「📋 テンプレートを使う」ボタンを押すと、あらかじめ用意された4種類のテンプレート（ダイエット／トレーニング／サプリ／生活習慣）が表示され、選択するとメモの下書きとして反映されます。ゼロから書く必要はありません。

### 🤖 AIチャット
- 自分が登録したナレッジベースの内容を踏まえて、AIが減量・ダイエットに関する質問に回答します（RAG方式）。
- 回答は300字程度に要約され、関連するナレッジがある場合は「📚 ナレッジ『○○』も参考に」という形で案内されます。
- AIの回答メッセージの下にある **「📝 ToDoに追加」** ボタンを押すと、回答内容が個別のタスクに自動で分割され、チェックボックスで選んだ項目（最大5件）を一括で減量ToDoリストに追加できます。

### 🌿 プロフィール（減量サポート用）
- 目標体重・目標日（年/月/日を個別に選択）などを設定します。

## 4. よくある質問（FAQ）

**Q. グループに参加できない**
A. グループ作成者から「参加ID」を共有してもらってください。自分のグループ一覧画面の「参加ID」横の「コピー」ボタンで共有用のIDをコピーできます。

**Q. AIの提案が出ない／エラーになる**
A. Gemini APIのクォータ超過時は自動的にルールベースの提案に切り替わります。しばらく時間を置いてから再度お試しください。解消しない場合は管理者に問い合わせてください。

**Q. 定期タスクが翌日も完了状態のままになっている**
A. タスク作成時に「🔄 定期タスク」のチェックを忘れている可能性があります。タスクの編集から設定を見直してください。

**Q. 体重やナレッジなどの個人情報は誰が見られる？**
A. 基本的に本人のみが閲覧できます。管理者ダッシュボードでは組織全体の集計データのみが可視化され、個人の体重ログやチャット内容そのものが直接表示されることはありません。

---

## 5. 注意事項

- 本アプリのAIによる提案・チャット回答は一般的な健康管理のアドバイスであり、医学的な診断・治療を目的としたものではありません。体調に不安がある場合は専門医・産業医に相談してください。
- 体重などの健康情報を入力する際は、画面を第三者に見せる場面に注意してください。
